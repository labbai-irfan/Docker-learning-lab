# Phase 8 — Security

Secure container images, supply chains, runtimes, and secrets.

---

## Topics

1. **Threat model & attack surface**
2. **Image scanning** — Trivy, Grype, Docker Scout, Clair, Snyk
3. **SBOM** — Syft, SPDX, CycloneDX
4. **Image signing** — Docker Content Trust (Notary), Cosign/Sigstore
5. **Supply chain security** — SLSA, provenance, attestations
6. **Runtime security** — Falco, Tracee, AppArmor, SELinux, seccomp
7. **Rootless containers**
8. **Least privilege** — cap drop, no-new-privileges, read-only FS
9. **Secrets management** — Vault, sealed secrets, BuildKit secrets
10. **CIS Docker Benchmark & compliance** — PCI-DSS, HIPAA, SOC2

---

## Learning objectives

✅ Can scan images for vulnerabilities  
✅ Can sign and verify images  
✅ Can generate SBOMs  
✅ Can write AppArmor/seccomp profiles  
✅ Can run rootless containers  
✅ Can audit container security  
✅ Can manage secrets in production  
✅ Can pass security compliance checks
