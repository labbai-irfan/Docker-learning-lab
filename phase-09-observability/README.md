# Phase 9 — Observability & Monitoring

Observe what's happening in production: metrics, logs, traces, alerts.

---

## Topics

1. **Metrics** — Prometheus, cAdvisor, node-exporter, dockerd metrics
2. **Visualization** — Grafana dashboards
3. **Logging** — log drivers, Loki, ELK/EFK stack, Fluentd
4. **Tracing** — OpenTelemetry, Jaeger, Tempo, Zipkin
5. **Alerting** — Alertmanager, PagerDuty
6. **Healthchecks & liveness**
7. **Golden signals** — latency, traffic, errors, saturation

---

## Learning objectives

✅ Can setup Prometheus + Grafana  
✅ Can write custom Prometheus queries  
✅ Can collect logs with Fluentd/Loki  
✅ Can instrument apps with OpenTelemetry  
✅ Can setup alerting  
✅ Can define meaningful dashboards  
✅ Can troubleshoot using metrics/logs/traces
