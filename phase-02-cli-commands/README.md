# Phase 2 — Docker CLI & Commands

Every Docker command, every flag, every hidden feature.

---

## Topics

1. **Lifecycle commands** — run, create, start, stop, restart, pause, unpause, kill, rm, wait, rename, update
2. **Inspection/Debug** — ps, logs, inspect, top, stats, events, port, diff, attach, exec, cp
3. **Image management** — build, buildx, image, images, pull, push, tag, rmi, save, load, import, export, commit, history
4. **Registry commands** — login, logout, search, scout
5. **Network commands** — network create/ls/inspect/connect/disconnect/rm/prune
6. **Volume commands** — volume create/ls/inspect/rm/prune
7. **System commands** — system df/prune/info/events, info, version
8. **Orchestration** — compose, swarm, service, stack, node, secret, config
9. **Advanced** — plugin, trust, manifest, context, builder, init

---

## Learning objectives

✅ Every Docker CLI command memorized with flags  
✅ Can read a `docker inspect` output and diagnose issues  
✅ Can use `docker exec` to debug a live container  
✅ Can build images with `docker build`, optimize for layers  
✅ Can push/pull from registries, understand auth  
✅ Can create/manage networks and volumes  
✅ Comfortable with compose, stack, swarm basics
