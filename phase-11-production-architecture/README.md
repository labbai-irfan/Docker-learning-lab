# Phase 11 — Production Architecture

Design systems that scale, survive failures, and cost less.

---

## Topics

1. **Architecture styles** — monolith, microservices, event-driven, SOA
2. **12-Factor App methodology**
3. **Statelessness & externalized config**
4. **High availability & redundancy**
5. **Disaster recovery** — RTO/RPO, backups
6. **Scaling** — horizontal, vertical, autoscaling
7. **Resource management** — limits, requests, QoS
8. **Multi-environment strategy** — dev, staging, prod
9. **Cost optimization** — FinOps for containers
10. **Capacity planning**
