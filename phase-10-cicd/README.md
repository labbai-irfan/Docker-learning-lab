# Phase 10 — CI/CD & Automation

Automate building, testing, scanning, and deploying containers.

---

## Topics

1. **GitHub Actions** — workflows, build/push/scan/sign
2. **GitLab CI/CD** — pipelines, runners
3. **Jenkins** — declarative pipelines, agents
4. **ArgoCD / Flux** — GitOps, continuous deployment
5. **Tekton, CircleCI, Drone** — other platforms
6. **Image build optimization** — BuildKit, layer caching, buildx
7. **Image promotion** — dev → staging → prod
8. **Registry workflows & retention** — lifecycle policies
9. **Deployment strategies** — rolling, blue-green, canary, A/B, shadow
10. **DevSecOps** — scan gates, policy as code (OPA)

---

## Learning objectives

✅ Can write GitHub Actions workflows  
✅ Can setup CI/CD pipelines  
✅ Can build/push/scan/sign images automatically  
✅ Can implement image promotion  
✅ Can deploy with GitOps (ArgoCD)  
✅ Can implement canary deployments  
✅ Can enforce security policies in CI/CD
