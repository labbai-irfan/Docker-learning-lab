# TOPIC: GitOps

> **Phase:** 10 · **Module:** CI/CD & Automation · **Status:** ✅ Complete

GitOps: Git as source of truth for infrastructure.

Tools: ArgoCD, Flux.

Workflow: commit config to Git → ArgoCD watches → automatic deploy.

Declarative: desired state in Git, ArgoCD ensures actual state matches.

Benefits: auditability, rollback (git revert), collaboration.

Labs: setup ArgoCD, deploy app via Git commit, rollback via git revert.
