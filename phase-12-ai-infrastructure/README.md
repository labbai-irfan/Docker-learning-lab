# Phase 12 — AI / ML Infrastructure

Deploy LLMs, RAG systems, and AI agents in containers.

---

## Topics

1. **GPU containers** — NVIDIA Container Toolkit, --gpus flag
2. **CUDA & cuDNN base images**
3. **LLM serving** — Ollama, vLLM, TGI (Hugging Face), llama.cpp, SGLang
4. **Models** — Llama, DeepSeek, Mistral, Qwen, Phi
5. **RAG stacks** — vector DBs (pgvector, Qdrant, Milvus, Weaviate, Chroma)
6. **Frameworks** — LangChain, LangGraph, LlamaIndex
7. **Agent systems & MCP servers in containers**
8. **Model packaging & multi-GB images**
9. **Inference scaling & batching**
10. **GPU scheduling** — MIG, time-slicing, fairness
